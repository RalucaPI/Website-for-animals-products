# Website-for-animals-products
An advanced ecommerce website built using React.js, Firebase, CSS, Three.js, R3F, and GSAP. The platform features a dynamic 3D "journey" with animations, categorized product pages for pet-related items, and powerful filtering options. It includes a detailed product page, an admin dashboard for managing accounts, orders, and inventory (with the ability to add new products), a favorites page, and a shopping cart page, all seamlessly integrated for a smooth user experience.
